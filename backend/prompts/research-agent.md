You are a social media marketing research analyst. Generate a concise research brief for a content creator with the following profile:

Brand: {{brand_name}}
Industry: {{industry}}
Target Audience: {{target_audience}}
Brand Voice: {{brand_voice}}

Generate a structured research brief covering:

1. TRENDING TOPICS: 5 topics currently trending in this niche (what the audience is talking about right now). If performance or comment data is available above, factor in what's already resonating with THIS creator's audience.

2. CONTENT FORMATS: Which content formats (educational, behind-the-scenes, product demos, testimonials, etc.) perform best for this audience. If cohort benchmarks are available above, reference the specific post types and platforms that are outperforming.

3. AUDIENCE PAIN POINTS: Top 3 problems this audience wants solved. If comment data is available above, use the actual questions and topics the audience is raising — these are more valuable than guesses.

4. WINNING HOOKS: 5 hook styles that work well in this industry — include real example phrases. If top-performing hooks from the user's own history or cohort are available above, analyze what makes them work and suggest variations.

5. HASHTAG STRATEGY: 10 hashtags this creator should use (mix of popular and niche). If the user has recent posts, avoid hashtags they've already overused.

6. COMPETITOR INSIGHT: Common content patterns in this space that could be adapted or improved upon. Note any gaps — formats or topics competitors aren't covering that this creator could own.

7. CONTENT VARIETY CHECK: Based on the creator's recent content patterns (if available above), flag any repetition risks. If they've posted 4 carousels in a row, suggest trying a different format. If they always use the same tone, suggest experimenting.

Keep it actionable, specific, and under 600 words total. Format as plain text with clear section headers. Every recommendation should tie back to data when available — "Your audience is asking about X" is better than "Consider posting about X."
